API Documentation for Movie API

List of Endpoints

Request URL Description
GET /movies Returns a list of all available movies.
Response Format:
[ { "id": 1, "title": "The Godfather", "year": 1972 }, ... ]
GET /movies/:title Returns data about a specific movie by title.
URL Parameter:
:title (movie's title)
Response Format:
{ "id": 1, "title": "The Godfather", "year": 1972, "description": "Movie description" }
GET /genres/:name Returns data about a genre (description) by name.
URL Parameter:
:name (genre's name, e.g., "Thriller")
Response Format:
[{ "Title": "Movie Title", "Genre": "Thriller" }]
GET /directors/:name Returns information about a director.
URL Parameter:
:name (director's name)
Response Format:
{ "name": "Francis Ford Coppola", "bio": "Director of The Godfather", "birthYear": 1939 }
POST /users/register Registers a new user.
Body Format:
{ "email": "user@example.com", "username": "newuser" }
Response Format:
{ "message": "User registered successfully!", "user": { "id": "1234", "email": "user@example.com", "username": "newuser" } }
PUT /users/:id Updates an existing user's information.
URL Parameter:
:id (user's unique ID)
Body Format:
{ "username": "updatedusername" }
Response Format:
{ "message": "User information updated successfully!", "user": { "id": "1234", "username": "updatedusername" } }
POST /users/:id/favorites Adds a movie to the user's favorites list.
URL Parameter:
:id (user's unique ID)
Body Format:
{ "movieId": 1 }
Response Format:
{ "message": "Movie has been added to your favorites!", "favorites": [...] }
DELETE /users/:id/favorites/:movieId Removes a movie from the user's favorites list.
URL Parameters:
:id (user's unique ID), :movieId (movie's unique ID)
Response Format:
{ "message": "Movie has been removed from your favorites.", "favorites": [...] }
DELETE /users/:id Deletes a user's account.
URL Parameter:
:id (user's unique ID)
Response Format:
{ "message": "User account has been removed successfully." }

Endpoint Details

Each endpoint corresponds to one of the actions defined in the API. Here are the main operations:

  • Return a list of all movies – Get a full list of movies available.
  • Return data about a single movie – Get detailed information about a specific movie by its title.
  • Return data about a genre – Get the description of a genre by its name (e.g., "Thriller").
  • Return data about a director – Get biography, birth and death year for a director.
  • Allow new users to register – Register a new user by providing an email and username.
  • Allow users to update their info – Update a user's username based on their ID.
  • Allow users to add a movie to their list of favorites – Add a movie to the user's favorites list.
  • Allow users to remove a movie from their list of favorites – Remove a movie from the user's favorites list.
  • Allow users to deregister – Delete a user from the system.
